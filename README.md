# myl_card_scanner_app

A Flutter card scanner application for Android and iOS.

## Getting Started

This Flutter project has been initialized with support for Android and iOS platforms only.

### Prerequisites

- Flutter SDK (3.0.0 or higher)
- For Android development: Android Studio and Android SDK
- For iOS development: Xcode (macOS only)

### Installation

1. Install Flutter dependencies:
```bash
flutter pub get
```

2. Run the app:
```bash
# For Android
flutter run -d android

# For iOS (macOS only)
flutter run -d ios
```

### Project Structure

This project supports:
- ✅ Android
- ✅ iOS

This project does NOT include:
- ❌ Web
- ❌ macOS
- ❌ Windows
- ❌ Linux

## Development

- `lib/` - Dart source code
- `android/` - Android native code
- `ios/` - iOS native code
- `test/` - Widget and unit tests
